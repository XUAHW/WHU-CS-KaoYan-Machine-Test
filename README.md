### 2014-T1 分段函数求值
编写程序，计算下列分段函数 $y=f(x)$ 的值。

根据输入的一个实数 $x$，按照以下公式计算 $y$：

$$
y = \begin{cases} 
\frac{\sqrt{x} + e^x}{5x + 5} & (x > 0) \\
\\
\left[ \frac{2}{3} + \sin(60^\circ) \right] \times |x| & (x \le 0)
\end{cases}
$$

**输入格式：**

* 输入包含一个实数 $x$。

**输出格式：**

* 输出一个实数 $y$，表示计算结果。结果保留小数点后 4位 (四舍五入)。

### 2014-T3 寻找完数
编写程序，显示 2 到 1000 之间（包括 2 和 1000）的所有完数。所谓完数，是指该数的各因子（真约数）之和正好等于该数本身。

例如：**6**的因子为 1, 2, 3（不包含 6 本身）。因为1 + 2 + 3 = 6，所以 $6$ 是一个完数。

**输入格式:**

* 无（本题为固定范围计算，不需要从键盘读取输入）

**输出格式:**

* 按从小到大的顺序输出。

* 数据之间用一个空格隔开，或者每个数占一行。

### 2014-T4 N阶魔方阵构造
N 阶魔方阵 是一个 $N \times N$ 的方阵，由自然数 $1, 2, \dots, N^2$ 组成。其特点是：每个元素值均不相等，且每行、每列以及主、副对角线上的 $N$ 个元素之和都相等。请根据以下罗伯法的算法规则，编写程序生成并输出一个 $N$ 阶魔方阵。

8&nbsp; 1&nbsp; 6

3&nbsp; 5&nbsp; 7

4&nbsp; 9&nbsp; 2  

**构造规则:**
1. 构造规则起始位置：数字 1 放置在第一行的正中间。
2. 移动方向：下一个数字的位置总是位于当前数字的右上方（即：行号减 1，列号加 1）。
3. 越界处理：上边界越界：如果行号超出上方边界（行号 < 0），则将其放入该列的最后一行。
4. 右边界越界：如果列号超出右方边界（列号 > N-1），则将其放入该行的最左一列。
5. 特殊情况（已被占位或 N 的倍数）：若当前数字是 $N$ 的整数倍，则下一个数字的位置直接选在当前位置的下方（行号加 1，列号不变）。

**输入格式:**

* 输入包含一个奇数整数 $N$ ($1 < N \le 21$)。(注意：该算法主要适用于奇数阶魔方阵)

**输出格式:**

* 输出 $N$ 行 $N$ 列的魔方阵

### 2015-T2 斐波那契数列中的质数
编写程序，生成斐波那契数列的前 30 个数，找出其中所有的质数，输出这些质数，并统计它们的个数。

**定义:**
1. 斐波那契数列 ($F_n$) 定义为：
   * $F_1 = 1$
   * $F_2 = 1$
   * $F_n = F_{n-1} + F_{n-2}$  (当 $n \ge 3$)
   * 即数列为：1, 1, 2, 3, 5, 8, 13, ...
2. 质数定义为：大于 1 的自然数，除了 1 和它本身外，不能被其他自然数整除。

**输入格式:**

* 无（本题为固定范围计算，不需要从键盘读取输入）

**输出格式:**

* 第一行：按从小到大的顺序输出前 30 个斐波那契数列中的所有质数，数字之间用空格隔开。
* 第二行：输出一个整数，表示这些质数的总个数。

### 2015-T4 整数分解
输入一个正整数 $n$，将其分解为若干个正整数之和。请编程输出所有可能的分解方案。

**规则要求:**
1. 分解出的数字之和必须等于 $n$。
2. 分解出的数字顺序应当从大到小排列（例如 4+2 是合法的，而 2+4 被视为与 4+2 重复，不予输出）。
3. 输出所有方案时，请按照字典序（第一位数字大的排前面）输出。

例如数字 $6$ 的分解包括：`6` `5+1` `4+2` `4+1+1`...以此类推。

**输入格式:**

* 输入包含一个整数 $n$ ($1 \le n \le 10$)。

**输出格式:**

* 每行输出一种分解方案。 数字之间用 + 连接。

### 2016-T2 倒杨辉三角
编写程序，从键盘输入一个整数 $n$，输出 $n$ 行的倒杨辉三角

**杨辉三角:**
1. 每一行的第一个数和最后一个数都是 1。
2. 从第三行开始，中间的每个数都等于它“两肩”上的数之和（即上一行同列位置与上一行前一列位置的数之和）。

倒杨辉三角即先输出第 $n$ 行，再输出第 $n-1$ 行……直到第 1 行。


**输入格式:**

* 输入一个整数 $n$（通常 $1 \le n \le 20$，以保证数据在整型范围内且屏幕能显示完整）。

**输出格式:**

* 第一行输出提示信息：please input a number: (注意冒号为英文冒号，且结尾无空格)。

* 紧接着输出 $n$ 行倒杨辉三角的数据。

* 每行中的数字之间用一个空格隔开。

* 每行数字左对齐显示。

### 2017-T4 统计与定位

* 第一行输出：`count = [次数]`
* 第二行输出：positions = [下标列表]
  * 下标之间用空格隔开。

**输出格式:**

* 输出格式为 `(x,y)`。

```
.<.<.
```
**样例1输出:**
```
(0,1)
```
