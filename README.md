### 2014-T1 分段函数求值
编写程序，计算下列分段函数 $y=f(x)$ 的值。

根据输入的一个实数 $x$，按照以下公式计算 $y$：

$$
y = \begin{cases} 
\frac{\sqrt{x} + e^x}{5x + 5} & (x > 0) \\
\\
\left[ \frac{2}{3} + \sin(60^\circ) \right] \times |x| & (x \le 0)
\end{cases}
$$

**输入格式：**

* 输入包含一个实数 $x$。

**输出格式：**

* 输出一个实数 $y$，表示计算结果。结果保留小数点后 4位 (四舍五入)。

### 2014-T2 字符串字符排序
从键盘输入一个字符串，请将此字符串中的所有字符按照 ASCII 码值 从小到大 的顺序进行排序，并输出排序后的字符串。

**输入格式:**

* 输入包含一行，为一个字符串。
* 字符串长度不超过 255。
* 字符串中不包含空格、制表符等空白字符。

**输出格式:**

* 输出一行，表示按 ASCII 码从小到大排序后的字符串。

### 2014-T3 寻找完数
编写程序，显示 2 到 1000 之间（包括 2 和 1000）的所有完数。所谓完数，是指该数的各因子（真约数）之和正好等于该数本身。

例如：**6**的因子为 1, 2, 3（不包含 6 本身）。因为1 + 2 + 3 = 6，所以 $6$ 是一个完数。

**输入格式:**

* 无（本题为固定范围计算，不需要从键盘读取输入）

**输出格式:**

* 按从小到大的顺序输出。

* 数据之间用一个空格隔开，或者每个数占一行。

### 2014-T4 N阶魔方阵构造
N 阶魔方阵 是一个 $N \times N$ 的方阵，由自然数 $1, 2, \dots, N^2$ 组成。其特点是：每个元素值均不相等，且每行、每列以及主、副对角线上的 $N$ 个元素之和都相等。请根据以下罗伯法的算法规则，编写程序生成并输出一个 $N$ 阶魔方阵。

8&nbsp; 1&nbsp; 6

3&nbsp; 5&nbsp; 7

4&nbsp; 9&nbsp; 2  

**构造规则:**
1. 构造规则起始位置：数字 1 放置在第一行的正中间。
2. 移动方向：下一个数字的位置总是位于当前数字的右上方（即：行号减 1，列号加 1）。
3. 越界处理：上边界越界：如果行号超出上方边界（行号 < 0），则将其放入该列的最后一行。
4. 右边界越界：如果列号超出右方边界（列号 > N-1），则将其放入该行的最左一列。
5. 特殊情况（已被占位或 N 的倍数）：若当前数字是 $N$ 的整数倍，则下一个数字的位置直接选在当前位置的下方（行号加 1，列号不变）。

**输入格式:**

* 输入包含一个奇数整数 $N$ ($1 < N \le 21$)。(注意：该算法主要适用于奇数阶魔方阵)

**输出格式:**

* 输出 $N$ 行 $N$ 列的魔方阵

### 2014-T5 硬币凑数
编写程序，使用递归方法计算：用 1 分、2 分和 5 分的硬币凑成 1 元（100 分），一共有多少种不同的凑法？

**规则说明:**
1. 目标金额为 1 元，即 100 分。
2. 可用的硬币面额为：1 分、2 分、5 分。
3. 每种硬币的数量不限。
4. 请尝试设计一个递归函数来完成计数逻辑（虽然循环也可以解决，但本题考察递归思维）。

**输入格式:**

* 无（本题为固定范围计算，不需要从键盘读取输入）

**输出格式:**

* 输出一个整数，表示共有多少种凑法。

### 2015-T2 斐波那契数列中的质数
编写程序，生成斐波那契数列的前 30 个数，找出其中所有的质数，输出这些质数，并统计它们的个数。

**定义:**
1. 斐波那契数列 ($F_n$) 定义为：
   * $F_1 = 1$
   * $F_2 = 1$
   * $F_n = F_{n-1} + F_{n-2}$  (当 $n \ge 3$)
   * 即数列为：1, 1, 2, 3, 5, 8, 13, ...
2. 质数定义为：大于 1 的自然数，除了 1 和它本身外，不能被其他自然数整除。

**输入格式:**

* 无（本题为固定范围计算，不需要从键盘读取输入）

**输出格式:**

* 第一行：按从小到大的顺序输出前 30 个斐波那契数列中的所有质数，数字之间用空格隔开。
* 第二行：输出一个整数，表示这些质数的总个数。

### 2015-T4 整数分解
输入一个正整数 $n$，将其分解为若干个正整数之和。请编程输出所有可能的分解方案。

**规则要求:**
1. 分解出的数字之和必须等于 $n$。
2. 分解出的数字顺序应当从大到小排列（例如 4+2 是合法的，而 2+4 被视为与 4+2 重复，不予输出）。
3. 输出所有方案时，请按照字典序（第一位数字大的排前面）输出。

例如数字 $6$ 的分解包括：`6` `5+1` `4+2` `4+1+1`...以此类推。

**输入格式:**

* 输入包含一个整数 $n$ ($1 \le n \le 10$)。

**输出格式:**

* 每行输出一种分解方案。 数字之间用 + 连接。

### 2015-T5 IP地址进制转换
编写程序，将输入的十进制点分形式的 IP 地址（IPv4）转换为二进制形式显示。

**转换规则:**
1. 输入格式为`A.B.C.D`，其中 A、B、C、D 均为 0 到 255 之间的整数。
2. 将每个十进制数转换为对应的 8 位二进制数。
3. 如果转换后的二进制数不足 8 位，需要在高位（左侧）补零。
4. 各段二进制数之间依然用小数点 . 连接。

**输入格式:**

* 输入包含一行字符串，表示合法的十进制 IPv4 地址。

**输出格式:**

* 输出一行字符串，表示转换后的二进制 IP 地址。

### 2016-T1 分解质因数
任何一个大于 1 的自然数，如果它是一个合数，都可以唯一地写成几个质数（素数）相乘的形式。请编写程序，输入一个正整数 $n$，将其分解质因数，并按照从小到大的顺序输出。

**例如:**
1. 输入 90，输出 `90=2*3*3*5`
2. 输入 7，输出 `7=7`


**输入格式:**

* 输入包含一个正整数 $n$ ($1 < n \le 2^{31}-1$)。

**输出格式:**

* 输出一行，表示分解质因数的结果。 等号右边的质因数必须按照从小到大的顺序排列，中间用乘号 * 连接。
